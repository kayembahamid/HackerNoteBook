# Solidity Storage Values Analysis

### Data Byte Sizes <a href="#data-byte-sizes" id="data-byte-sizes"></a>

Reference: [https://tomatosauce.jp/datatype-bytesize/](https://tomatosauce.jp/datatype-bytesize/)

In Solidity, each data type has the following size:

| Type           | Bytes |
| -------------- | ----- |
| bool           | 1     |
| bytes1         | 1     |
| bytes8         | 8     |
| bytes16        | 16    |
| bytes32        | 32    |
| address        | 20    |
| contract       | 20    |
| uint8/int8     | 1     |
| uint16/int16   | 2     |
| uint32/int32   | 4     |
| uint64/int64   | 8     |
| uint128/int128 | 16    |
| uint256/int256 | 32    |
